The Grimoire
Dark, arcane, atmospheric. Gold-on-dark with parchment textures, ornamental borders, and Cinzel serif typography. Feels like opening an ancient spellbook. Vignette and glow effects create dramatic mood.
Dark theme Serif-heavy Atmospheric Ornamental Fantasy immersion

A+
The Grimoire (High Contrast)
Same arcane atmosphere, now WCAG 2.1 AA compliant. Every text color ≥ 4.5:1, non-text UI ≥ 3:1. Brighter golds (#c9a54a, #ddb84e), warmer crimson (#ef5350). Adds skip link, focus-visible styles, semantic landmarks, ARIA labels, and prefers-reduced-motion support. Vignette softened to preserve edge contrast.
WCAG AA Dark theme Accessible Skip link Reduced motion
↗
B
Obsidian Arsenal
Modern dark UI with sidebar navigation, accent colors for data types (HP red, AC teal, initiative gold). Instrument Serif headlines + JetBrains Mono data. Clean, app-like, information-dense — closest to DnDBeyond's sensibility but sharper.
Dark theme App-like Sidebar nav Data-forward Modern
↗
C
The Warm Tavern
Light, warm, inviting. Cream backgrounds with rich wood-brown and ember accents. Fraunces variable serif headlines, generous border-radius, card-based layout. Friendly and approachable — feels like a beautifully illustrated handbook.
Light theme Warm palette Card layout Rounded Inviting
↗
D
Ironworks
Industrial utilitarian. Light background, black top bar, square corners, grid-locked panels with shared borders. Archivo bold sans + Literata serif names + Fira Code mono data. Maximum information density, zero ornamentation — the spreadsheet-player's dream.
Light theme Utilitarian Grid-locked Dense No-nonsense
